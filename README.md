# Miraj Ud Din - AI Engineer Portfolio

Welcome to the repository for my personal portfolio website! This project is a fully responsive, modern web application designed to showcase my academic background, technical skills, and practical machine learning projects. 

## 🌟 Overview

I am an Artificial Intelligence undergraduate at the **University of Agriculture, Peshawar**, passionate about turning raw datasets into intelligent, production-ready systems. This portfolio highlights my journey in integrating academic research with practical AI solutions.

You can view live deployments of my projects, including comparative regression studies, classification models, and more.

## 🚀 Features

- **Modern UI/UX**: Designed with a sleek, dark-themed aesthetic, glassmorphism elements, and vibrant, eye-catching gradients.
- **Fully Responsive**: Flawless experience across desktop, tablet, and mobile devices.
- **Dynamic Animations**: Smooth scroll reveals, hover effects, and CSS transitions to enhance interactivity.
- **Interactive Capabilities**: Features a working contact form and links to my live project deployments (e.g., Streamlit applications).
- **Comprehensive Sections**: 
  - About Me & Education
  - Technical Toolkit (Languages, Libraries, & Tools)
  - Project Showcase (with metrics and links)
  - Publications & Certifications
  - Extra Activities & Interests

- **Frontend**: HTML5, Vanilla CSS3 (Custom Design System, Flexbox/Grid), JavaScript (DOM Manipulation, Scroll Observers)
- **Icons & Graphics**: FontAwesome, Devicons, Custom SVG Assets
- **Deployment**: Configured for static hosting (GitHub Pages, Vercel, Netlify)

## 💻 Setup & Installation

This project is a static website and requires no complex build tools or backend to run locally.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/mirajuddin1357/Portfolio.git
   ```
2. **Navigate into the directory:**
   ```bash
   cd Portfolio
   ```
3. **Run the site locally:**
   Simply open the `index.html` file in any modern web browser. Alternatively, you can run a local development server using Python:
   ```bash
   python -m http.server 8000
   ```
   Then navigate to `http://localhost:8000` in your browser.
